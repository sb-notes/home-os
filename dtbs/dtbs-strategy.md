# Boots — Strategy & Context

---

## About Boots

- **Company:** Boots
- **Size:** ~10 people
- **Location:** UK
- **Founder/CEO:** Fern
- **Original model:** Data migration consultancy — helping organisations migrate and manage data pipelines
- **Strategic pivot:** Built internal SaaS tooling to make their own assessments faster and more rigorous. That tool is now a standalone platform ready for external launch.

---

## The Platform

Boots built assessment tooling because they needed it themselves. Consultants used it to evaluate an organisation's data pipelines and migration requirements more efficiently. It has since been developed to the point where it can be sold to other consultancies and professional services (PS) firms as a licensed SaaS platform.

**Platform function:** Data and analysis tooling for data migration project assessments — enables structured, repeatable, efficient assessment of an organisation's data estate, readiness, and migration requirements.

**Status:** Built. Ready to launch externally.

---

## Strategic Positioning — Core Anchor

**"Snowballing Data Pain in transformation"**

Data migration is underestimated in every major transformation project. The pain compounds — poor assessment at the start causes exponential problems downstream. Boots' platform addresses this by making rigorous data assessment fast, repeatable, and accessible to any consultancy.

This "why" is the agreed anchor. It resonates because it's true to every PS consultant who has lived through a botched migration — and it positions the platform as a risk-reduction tool, not just an efficiency tool.

## The Dual-Audience Problem (unresolved)

Boots has two fundamentally different buyer types:

**Buyer A — End clients (organisations)**
- They need a data migration assessment done
- They buy the five-day assessment offer (entry point)
- They don't care about the platform — they care about the outcome
- Decision-maker: CTO, CIO, Head of Data, Transformation Director

**Buyer B — Consultancy partners / PS firms**
- They want to license the platform to run their own assessments
- They care about speed, repeatability, and differentiating their own offering
- Decision-maker: Practice lead, MD, Head of Delivery
- They are also potential channel partners — each consultancy that licenses the platform multiplies Boots' reach

**The tension:** Positioning and GTM are different for each. A message that resonates with a CTO looking to commission an assessment won't land the same way with a consultancy practice lead evaluating a tool to license.

**Status:** Not yet resolved. Next session should explore: unified positioning with audience-specific proof points, or separate tracks?

## The Five-Day Assessment Offer

The planned low-risk entry point for end clients — a structured, time-boxed data migration readiness assessment delivered in five days using the platform. Designed to:

- Reduce friction for new clients (defined scope, defined price, defined output)
- Showcase the platform's capabilities in a real engagement
- Convert assessment clients into ongoing platform or consultancy clients

**Status:** Concept established. Needs refinement as a product-led entry point — the offer should be framed around the platform's output quality, not just the consultancy's expertise.

## GTM Thinking

### For end clients

- Target: organisations undergoing or planning a major data migration (ERP rollouts, cloud migrations, M&A integrations)
- Entry: five-day assessment offer — easy to say yes to
- Channels: housing sector (conference traction), financial services, NHS/public sector (common large-scale migration context)
- Proof points: before/after on assessment speed and accuracy vs manual approaches

### For consultancy partners

- Target: mid-sized data migration and transformation consultancies (10–200 people) who do this work but lack assessment tooling
- Entry: platform demo + pilot engagement
- Channel: direct outreach to practice leads, conference presence, referrals from end client assessments
- Proof points: time saved per assessment, repeatability, white-label or co-branded options (TBC)

### Housing sector — early traction

- Fern had a booth demo at a UK housing conference
- Positive feedback received
- Housing associations and local authorities are frequent data migration clients (legacy systems, cloud migrations, mergers)
- Could be a beachhead vertical if traction continues
