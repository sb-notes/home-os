### Wispr Flow / Gboard integration — Pixel 8 (4 Apr 2026)

- Wispr Flow icon periodically disappears from Gboard toolbar after restart
- Root cause: Android adaptive battery / background permission stripping
- **Fix applied:** Battery set to Unrestricted (Settings → Apps → Wispr Flow → Battery); background usage enabled; "Manage app if unused" confirmed disabled (greyed out)
- Floating overlay bubble dismissed; Gboard toolbar icon confirmed as preferred trigger
- **Volume button shortcut** (hold both volume keys simultaneously) configured as fallback if toolbar icon disappears
- Accessibility icon turned off to reduce clutter
- **Known residual issue:** Icon can reappear missing after OS updates — workaround: toggle Wispr Flow off and back on before each journal transcription session. Battery Unrestricted may need re-checking after OS updates.
