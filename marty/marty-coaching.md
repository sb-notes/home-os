## Key frameworks in use

**SHARP interview framework (Alex Rechevskiy):**  
Hook → Principles → Actions → Results → Learnings. Use for probation review stories, future PM interviews.

**7W AI Product System:**  
Treat AI as product capability, not feature. 8 automation candidate types mapped as discovery backlog.

**Value → UX → UI funnel:**  
Validate value before designing experience; design experience before polishing interface. Pre-PMF discipline.

**Assembly-first discipline (OneBrain):**  
Build with what the platform already supports before speccing new engineering work. Validate before building.
